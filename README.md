How to start application:
`flask --app app/routes.py run --host=localhost --port=5174`
"-p" for port

Python setup & env:
1. Create and activate virtual env.
`python3 -m venv venv`

2. Activate virtual env.
Linux:
`source venv/bin/activate`
Windows cmd:
`venv\Scripts\activate`

3. Install setup.py
`pip install -e .`
